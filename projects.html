Here's a collection of projects that showcase my journey as a computer engineering student. Through my internship at New York ITS, I gained hands-on software engineering experience working on critical state infrastructure systems. I developed automation tools using PowerShell and SQL, created web applications with C# and Razor Pages, and optimized database performance by consolidating multiple legacy functions into a single, more efficient solution.

The projects below demonstrate my evolution as an engineer—from early experiments with hardware and embedded systems to more sophisticated software solutions that solve actual problems for users.
